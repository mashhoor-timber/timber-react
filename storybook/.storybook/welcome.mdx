# Welcome to Timber Component Library for React

---

## üöÄ Getting Started

The Timber Component Library is a comprehensive React component library designed to help you build beautiful, functional applications quickly and efficiently. Our components are built using modern React patterns, TypeScript support, and accessibility best practices.

**üí° Build Tool Recommendation:** We recommend using <a href="https://vite.dev/guide/" target="_blank" rel="noopener noreferrer">Vite</a> as your build tool for React projects. Vite provides lightning-fast development server startup, instant hot module replacement (HMR), and optimized builds, making it an excellent choice for modern React development.

### Installation (use yarn or npm, whichever you prefer)

```bash
yarn add timber-react
```

OR

```bash
npm install timber-react
```

### Required Dependencies (use yarn or npm, whichever you prefer)

Install these packages alongside Timber:

```bash
yarn add framer-motion@12.23.0
```

```bash
yarn add tailwindcss@3.4.13
```

```bash
yarn add @heroui/system@2.4.7
```

```bash
yarn add @heroui/theme@2.4.6
```

```bash
yarn add autoprefixer@10.4.13
```

```js
module.exports = {
  plugins: {
    tailwindcss: {},

  },
};
```

### Usage

```tsx
import { TimberProvider, AddExpenseForm } from 'timber-react';

const config = {
    apiKey: <api_key_here>
}

function App() {
  return (
    <TimberProvider config={config}>
      <AddExpenseForm onSubmit={(data) => console.log(data)} />
    </TimberProvider>
  );
}
```
