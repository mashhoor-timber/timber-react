# Welcome to Timber Component Library for React

---

## 🚀 Getting Started

Timber Component Library is a comprehensive React component library designed to help you build beautiful, functional applications quickly and efficiently. Our components are built with modern React patterns, TypeScript support, and accessibility in mind.

### Installation

```bash
yarn add timber-react-dev
```

### Usage

```tsx
import { TimberProvider, AddExpenseForm } from 'timber-react-dev';

const config = {
    apiKey: <api_key_here>
}

function App() {
  return (
    <TimberProvider config={config}>
      <AddExpenseForm onSubmit={(data) => console.log(data)} />
    </TimberProvider>
  );
}
```
